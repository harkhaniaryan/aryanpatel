# what is web server ?

a web server is a system that stores websites data and delivers it to users when they request it through a web browers.

**web servers works**

1. you entre a websites URL in a browser.
2. the browser sends a request to the web server.
3. the web server processes the request.
4. it sends back the webpage (HTML,CSS,IMAGES)
5. the browsers diplays the webpage.

**functions of a web browsers**

1. stores website files
2. handles user requests
3. sends web pages to browsers
4. manages website traffic

# examples

1. apache HTTP server
2. nginx
3. microsoft IIS
