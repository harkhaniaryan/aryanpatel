# what is web hosting

web hosting is a srevice that allows individuals or organization to store their website on the internet so that people can acess it anytime.

**web hostings works**
1.you create a website. 2. you upload website files to a hosting server. 3. when someone enters your websites URL the server sends the website to their browsers. 4. the website is displayed on their screen.

**types of web hosting**

1. shared hosting-many websites share one server.
2. VPS hosting-virtual private server with more control.
3. dedicated hostong-one server for one website.
4. cloud hosting -uses multiple servers for better performance.
