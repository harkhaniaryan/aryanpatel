# what is protocol

**Key features of protocols**

1. Format – How data is structured
2. Transmission – How data is sent
3. Error handling – What happens if data is lost
4. Synchronization – Timing of communication

# Common types of protocols

1. HTTP / HTTPS
   Used for websites
   HTTPS is secure (encrypted)
2. FTP (File Transfer Protocol)
   Used to transfer files between computers
3. TCP/IP
   The core protocol of the internet
   Ensures data is delivered correctly and in order
4. SMTP
   Used to send emails
5. DNS (Domain Name System)
   Converts website names into IP addresses

6. browsing on Google Chrome

When you send a message on WhatsApp:

1. Your phone follows certain rules to send the message
2. The internet follows rules to deliver it
3. The receiver’s phone follows rules to display it

These rules together are called protocols.
