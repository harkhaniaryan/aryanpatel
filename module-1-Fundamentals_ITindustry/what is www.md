# what is www ?

the world wide web is a system of interlinked web pages and resources that you can access over the internet using a web browsers.

**www works**

1. you enter a website address URL in a browsers.
2. the browsers sends a request over the internet.
3. a web server sends back the webpaeg.
4. the browsers displays the page.

**components of WWW**

1. web pages (documents on the internet)
2. web browsers (to view pages)
3. web servers (to store and send pages)
4. URLS (adresses of web pages)

**importance of www**

1. helps in communication and information sharing.
2. used for education,business,entertainment.
3. makes the internet user-friendly.
